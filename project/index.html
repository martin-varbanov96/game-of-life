Game of life

Rules:

In this game there are a number of squares. The brown ones are the dead, the green ones are alive. As time passes some die, some are born.
If a living square has 2 or 3 living neighbours, it will keep on living, else, it dies.
If a dead square has exactly 3 living neighboors, after the step, it will be brought back to life.

How to play:

Set the size of the board the way you like. Chose which square is going to be alive and which- dead, click STEP INTO to see what will happen in one step time, click STEP BACK to see what will happen one step back.
